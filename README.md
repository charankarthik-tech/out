# out

A simple static site that lists educational links under the heading **Open Source Learning Webpage**.  

This repository can be hosted permanently using [GitHub Pages](https://pages.github.com/).

**Live app URL (once deployed):**
https://charankarthik-tech.github.io/out/

## Development

Edit the HTML, CSS, or `links.js` to add or remove resources. Commit and push to the `main` branch; the GitHub Pages workflow (see `.github/workflows/gh-pages.yml`) deploys the site automatically.
